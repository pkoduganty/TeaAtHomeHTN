Detailed Description

Properties of the robot arm.

All items the robot can carry with his arm have a corresponding enum field in RobotArm. Because of a dynamic number of teacups, the teacup fields are created programmatically and RobotArm is recreated during runtime. E.g. RobotArm.teacup1, RobotArm.teacup2, ..., RobotArm.teacup75
