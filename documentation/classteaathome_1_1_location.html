Detailed Description

Properties for items which can be moved.

We do NOT use a hierarchy of locations (e.g. multiple shelfs in a cupboard). It is implicitely given, that shelf 1-4 are in cupboard 1 and shelf 5-8 are in cupboard 2. Exception: We declare the teabag as "Location.inteacup" as simplified solution.
